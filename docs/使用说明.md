# 🎨 图片元素分割工具 - 快速指南

## 📋 四个脚本的用法

### 1️⃣ **paste_and_split.py** ⭐ 最快
```
直接复制图片 (Ctrl+C) → 运行脚本 → 自动保存 + 分割
双击运行或：python paste_and_split.py
```
**适合场景：** 截完一张图直接处理

---

### 2️⃣ **gui_splitter.py** ⭐ 最方便
```
图形化窗口界面，点按钮选择文件
已运行中 → 选择图片 → 自动打开输出文件夹
```
**适合场景：** 不想记命令，直观操作

---

### 3️⃣ **quick_split.py**
```
自动处理 D:\游戏开发\参考图片 文件夹中的所有PNG
python quick_split.py
```
**适合场景：** 已经把所有PNG放进文件夹，一键全处理

---

### 4️⃣ **crop_transparent.py**
```
只去透明边，不分割元素
python crop_transparent.py
```
**适合场景：** 只需要简单裁裁，不需要分割

## 🔧 功能说明

### 去透明边
- 自动检测Image的四周透明区域
- 智能裁剪，保留所有非透明内容
- 保持原有的透明度信息（RGBA）

### 元素分割
- 使用连通区域标记算法
- 自动识别独立的对象
- 为每个元素生成独立文件
- 每个元素自动添加小padding确保完整性

## 💡 使用技巧

### 批量复制：
```
1. 在文件夹里 Shift+点击 选择多个文件
2. Ctrl+C 复制全部
3. 在 D:\游戏开发\参考图片 里 Ctrl+V 粘贴
4. 运行 quick_split.py
```

### 处理不当？
- PNG出现黑色背景？→ 原图就是黑色，脚本工作正常
- 分割数量奇怪？→ 复杂纹理可能有很多连通区域，正常现象
- 完全没裁掉？→ 可能图片完全没透明边，已经是最优

## 🐛 遇到问题？

### 脚本不运行：
```powershell
# 检查Python是否安装
python --version

# 重新安装依赖
pip install Pillow scipy numpy
```

### 输出文件夹找不到：
```
D:\游戏开发\参考图片_elements
# 会自动在参考图片文件夹里生成，名字是 [原图名称]_elements
```

### 元素太多/太少：
- 这取决于图片的复杂度和阈值
- 脚本使用的是 Alpha > 128 来判断透明/不透明
- 可以修改参考fcode找 `> 128` 来调整敏感度
